# MyCalendar v1.0.0

这是 MyCalendar 的第一个可发布版本。

## 主要功能

- 日视图、周视图、月视图
- 小窗模式和全屏模式
- 本地 JSON 保存日程
- 日视图时间轴展示
- 拖选时间段快速创建日程
- 点击日程编辑、完成、取消或删除
- 周视图小窗横向滚动
- 月视图用彩色圆点和横条展示日程
- 日期悬浮轮盘
- 返回上一步和回到今天

## 使用方式

下载 `MyCalendar.exe` 后直接双击运行。

程序会在 exe 所在目录旁边自动创建：

```text
data/events.json
```

这个文件用于保存本地日程数据。

## 注意

- 这是一个个人本地日历工具，不包含账号登录和云同步。
- Windows 第一次打开时可能提示未知发布者，这是因为 exe 没有代码签名。
- 如果要保留日程数据，请不要删除 exe 旁边的 `data/events.json`。
